## Relay Strategies

### Strategy 1: Speed (Fastest Load Times)
Use only 2-3 fastest relays:
1. Disable all default relays
2. Add: `wss://nos.lol` (fast, reliable)
3. Add: `wss://relay.damus.io` (Damus - popular)
4. Keep disabled: Nostr.band (slower for discovery)

**Result**: Fastest feed, might miss some content

### Strategy 2: Discovery (Most Content)
Use all available relays for maximum coverage:
1. Enable all 4 default relays
2. Add 2-3 additional relays you trust
3. Accept slightly slower loading

**Result**: Maximum content, comprehensive feed

### Strategy 3: Privacy (Maximum Decentralization)
Use diverse, smaller relays:
1. Disable popular relays (Damus, Snort)
2. Add smaller, community-run relays
3. Examples: Personal nodes, small communities

**Result**: More privacy, support decentralization

### Strategy 4: Stability (Most Reliable)
Use only the most stable relays:
1. Keep: Damus, Snort (battle-tested)
2. Remove: Lesser-known relays
3. Add backup: Nostr.band

**Result**: Reliable connection, proven uptime

## Power User Tips

### Tip 1: Multiple Logins (Advanced)
1. Open app in normal window (nsec login)
2. Open app in private/incognito window (different npub)
3. Compare feeds or monitor multiple accounts

**Note**: Each window has separate localStorage

### Tip 2: Relay Testing
Before adding a relay permanently:
1. Add the relay
2. Wait 30 seconds for connection
3. See if content loads
4. Remove if unstable
5. Add permanently if working

### Tip 3: Custom Relay Sources

Find new relays at:
- https://nostr.watch - Relay monitoring
- https://nostr.band - Relay explorer
- https://nostr-relay-cli.github.io - Relay list
- Nostr client lists (check your favorite client)

### Tip 4: Troubleshooting Feed Issues

**Feed is slow?**
1. Disable 1-2 relays temporarily
2. Check internet speed (speedtest.net)
3. Try closing other apps
4. Refresh page

**Not seeing content?**
1. Make sure at least 2 relays are enabled
2. Verify relays are online (check nostr.watch)
3. Add more hashtags to search
4. Wait a minute for content to load

**Getting duplicates?**
1. Click Refresh button
2. Remove and re-add relays
3. Clear browser cache (Ctrl+Shift+Delete)

## Security Reminders

### Never
- ❌ Share your nsec online (it's like a password)
- ❌ Paste nsec in browser console
- ❌ Give nsec to anyone claiming to be support
- ❌ Type nsec in public places
- ❌ Store nsec in plain text files

### Always
- ✅ Keep nsec backed up securely (password manager)
- ✅ Use npub for sharing (public key)
- ✅ Logout before leaving public computers
- ✅ Use strong computer password
- ✅ Keep browser updated

## Troubleshooting Power Tips

### Advanced Troubleshooting

**App is completely broken?**
1. Open browser DevTools (F12)
2. Go to Console tab
3. Check for error messages
4. Screenshot the error
5. Report on GitHub

**localStorage is corrupted?**
1. Open DevTools (F12)
2. Go to Application tab
3. Click localStorage
4. Right-click and delete
5. Reload page

**Relay keeps disconnecting?**
1. Check relay status: nostr.watch
2. Try different relay
3. Check your internet
4. Restart browser
5. Try different device
